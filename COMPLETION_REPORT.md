┌──────────────────────────────────────────────────────────────────┐
│ 🚀 快速开始指南
└──────────────────────────────────────────────────────────────────┘

【第一步】安装 Qt 环境
  1. 访问 https://www.qt.io/download
  2. 下载 Qt 5.15+ (Windows)
  3. 在安装时选择 MSVC 2019/2022 编译器

【第二步】验证环境
  • 打开命令提示符
  • 输入 qmake --version 验证 Qt
  • 输入 nmake --version 验证编译器

【第三步】编译项目
  方式 A (自动脚本):
    • 双击 build.bat 文件
    • 选择编译类型 (Debug/Release)

  方式 B (手动命令):
    • 打开命令提示符
    • cd 到项目目录
    • qmake AutoFileCleaner.pro -spec win32-msvc
    • nmake release

【第四步】运行应用
  • 双击 run.bat 文件
  • 或直接运行 build/release/AutoFileCleaner.exe

【第五步】创建任务
  1. 点击"新建任务"按钮
  2. 填写任务名称和描述
  3. 点击"添加文件夹"选择目标路径
  4. 设置清理阈值 (多少天未访问)
  5. 设置执行时间 (每天几点运行)
  6. 点击"确定"保存
